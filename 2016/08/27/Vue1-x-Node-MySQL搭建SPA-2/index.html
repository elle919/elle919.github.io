Vue1.x+Node+MySQL搭建SPA(2)

发表于 2016-08-27

配置MySQL数据库

在MySQL官网上下载并安装MySQL server，创建用户admin，然后用admin账户登录

创建数据库prod_database

然后创建表house

也可以下载MySQL workbench，直接操作创建表，不用自己写sql语句

连接数据库

配置完MySQL之后，就要在node环境里连接数据库，来操作数据库增删改查了
首先安装mysql模块

安装完之后就可以用这个模块与mysql进行交互了，我们创建db.js文件，这样执行node db就可以连接数据库了

安装express

先安装express，express是目前最流行的基于Node.js的Web开发框架，可以快速地搭建一个完整功能的网站。

然后我们创建app.js，将db相关的配置放在db.js里，然后在app.js里引入；可以将所有请求都放app.js里头写，但是这样太ugly了，所以我们抽取出去，看最后一行代码app.use(‘/houses’, require(‘./src/controllers/houses’))，就是将需要处理的逻辑放在了’./src/controllers/houses’里头了
app.use是用来加载用于处理http请求的中间件，在/houses路径中任何类型的http请求都会执行后面的require(‘./src/controllers/houses’)函数

db.js，我们用createPool的方法来创建数据库，createPool比createConnection性能更好些

构建MVC架构

先看一下我们的目录结构，在src下增加views、controllers、models目录，views放所有的页面，controllers处理所有http请求，models写sql语句取数据

Controllers

在上一节中，我们在app.js里用了app.use(‘/houses’, require(‘./src/controllers/houses’))，现在我们看看controllers/houses里的代码，可以看到router处理/houses/list路径的get请求，获取房屋列表的值，然后调用了model层houseModel.getAll的方法

Models

在model层中我们主要连接数据库，并写sql语句去增删改查prod_database里的数据

Views

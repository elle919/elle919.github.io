搭建vue环境

vue-cli安装vue环境

现在使用Vue特别简单，官方提供了vue-cli，可用于快速搭建大型单页应用。

npm run dev其实是package.json里头的缩写，打开package.json可以看到项目的名称、版本号、描述、作者等信息，scripts则是对某些命令的简写，执行npm run dev时，实际上执行的是node build/dev-server.js

执行完npm run dev后，在浏览器打开localhost:8080就可以看到页面，这里因为我是在服务器10.16.111.13上重新部了下系统所以输入的是10.16.111.13:8080，写文章的时候vue-cli已经出了2.x版本，如果用的是旧版本的话会有提示

webpack默认用eslint检查代码风格，导致总是在报warning，空格数、分号等都有严格的限制，看个人风格吧，觉得太麻烦的话可以在/build/webpack.base.conf.js里头将关于eslint的preloader给注释掉：

安装vue-router和vue-source

好了，我们的项目已经可以跑起来了。对于单页应用，我们还需要安装官方库vue-router，以及用来和服务器通信的vue-resource

进入src/main.js，引入安装的router和resource等，并用路由指定跳转页面，参见vue-router使用手册 http://router.vuejs.org/zh-cn/installation.html

进入src/App.vue，写一下页面布局，用v-link属性指定跳转的链接，<router-view>标签则是将router里指定的component渲染到这里

再看index.html

这样就实现了单页应用的路由功能了，新写的页面只要通过src/main.js里的router配置就能指定相关页面了。
